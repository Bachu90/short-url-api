API Project: URL Shortener Microservice

How to use:

  1. You can add a new URL via POST method using http://grzegorzbach-short-url.herokuapp.com/api/shorturl/new endpoind or a form below, and you will receive a shortened URL in the JSON response.
  2. You can add a new URL via GET method using http://grzegorzbach-short-url.herokuapp.com/api/shorturl/new?url={your url} endpoind, and you will receive a shortened URL in the JSON response.
  3. If you pass a valid URL that follows the http(s)://www.example.com(/more/routes) format, the JSON response will look like this: {"original_url":"www.google.com","short_url":1},
    in any other case you will get an error object response: {"error":"invalid URL"}
  4. When you visit the shortened URL (which already exists in api database) using: http://grzegorzbach-short-url.herokuapp.com/api/shorturl/{short url} endpoint , it will redirect you to its original link.
  5. You can GET a list of all URL`s from http://grzegorzbach-short-url.herokuapp.com/api/allurls
